Синтаксис

JQuery - это библиотека JavaScript, которая фокусируется на его взаимодействие с HTML.
Библиотека позволяет легко получить доступ к любому элемету DOM, атрибутам и содержимому.
Простое боращение к h1 для изменения цвета текста выгладит так:
jQuery("h1").css("color","green");
Вместо слова jQuery можно написать $ ($("h1").css(...)), такая запись будет полностью идентична.
& - это синоним слова jQuery.

Если CDN подключен не в конце файла (внизу body), то необходимо использовать метод .ready
$(document).ready(function(){
$("h1").css("color", "green");
});
Сокращенный вариант без явного вызова метода .ready объекта document:
$(function(){
$("h1").css("color", "green");
});

Для добавления\изменения текста содержимого элемента, вместо метода .css используется .html
$("h1").html("new text");//в h1 будет записано "new text"

В следующем примере с помощью кнопок на странице будет появляться и исчезать блок текста.
Обращаться к DOM элементам будем по идентификатору (id="") с помощью $('#btn_start').
$('#btn_start').click(function(){
$('#message')// последующие 6 методов добавляют свойства блоку #message
.html("Lorem ipsum")
.css("background-color", "green")
.parent()
.css("background-color", "#fff4dd")
.width(500)
.height(80)
});
$('#btn_reset').click(function(){
location.reload();
});

Выбор селекторов. Оптимизация выборки.

Доступ к элементу по идентификатору (id):
$("#content").css(...);
Доступ к элементу по атрибуту класс (class):
$(".wrapper").html(...);
Можно комбинировать селекторы в jquery запросе:
$("#content .wrapper").css(...);
$(".box .wrapper").html(...);
Доступ к элементу по тегу (<...>):
$("span")...;
Селекторы по тегу можно перечислять через запятую:
$("h1,h3")...;
Вложенные теги (<div> <article> </article> </div>) можно прописать через пробел. Выберем все теги article внутри div:
$("div article")...;
Есть другой способ выбрать вложенный тег - с помощью метода .find:
$("div").find("article")...;
Доступ к соседнему элементу (вроде как в css). Выберем тег "p", который идет сразу за тегом "h2":
$("h2 + p")...;
Выбор элементов - потомков (определенных или всех вложенных элементов):
$("article > h2")...// выберет ВСЕ h2 внутри всех article
$("article > *")...// выберет ВООБЩЕ ВСЕ элементы вложенные в article

В jQuery есть два метода похожие на :BEFORE и :AFTER из CSS - это .prev() и .next():
Допустим у нас есть div#content, перед ним на том же уровне стоит header, а после него стоит footer:
<header> </header>
<div id="content"> </div>
<footer> </footer>
$("#content").prev().css("...");// выберет элемент header
$("#content").next().html("...");// выберет элемент footer
Чтобы выбрать ВСЕ элементы, используется *:
$("*").css(...);

Оптимизация выборки

У нас есть такой запрос:
$("#content a.button")...;
Использование метода .find для выбора вложенных элементов ускорит работу:
$("content").find("a.button")...;
Также удобно разбивать запрос на более мелкие и простые составные части.
Для этого используется "контекст". Оптимизируем два следующих запроса:
$(".content a.button");
$(".content h3.title");
Создадим переменную "content":
var $content = $(".content")
Затем используя переменную находим вложенные элементы:
$content.find("a.button");
$content.find("h3.title");

Работа с CSS. Изменения атрибутов элемента.

Для изменения стилей элементов используется метод .css(). Свойство\значение можно задавать через запятую или двоеточие(:).
#(".content").css("background-color", "green");
Чтобы за один запрос установить несколько стилей используется {} и свойства перечисляются через запятую:
$(.box).css({'color':'red', 'font-size':'12px', 'margin-left':'50px'})
Значение свойств ('12px') необходимо указывать в кавычках всегда, но сами свойства можно писать без кавычек, при условии что тире (-) не ставится и следующее словоначинается сразу и с заглавной буквы (background-color будет backgroundColor):
$(.box).css({color:'red', fontSize:'12px', marginLeft:'50px'})

Также манипулировать CSS можно функцией обратного вызова (значение свойства задается функцией):
$("#content").css('height', function(i, value){
return parseFloat(value)*1.2;
})

Манипуляции с классами

Метод .addClass() позволяет добавить атрибут класс какому-либо элементу:
$("#box").addClass('wrapper');
Проверить причастность элемента к какому-либо классу можно методом .hasClass.
При добавлении класса также можно использовать функции обратного вызова:
$("#content").addClass(index, function(...){ return className;...})
Удаление атрибута класс у элемента производится методом .removeClass()
$("#content").removeClass('box');
.removeClass часто используют для создания выпадающих меню.
Переключение класса производится методом .toggle()
$("h2").toggle();//Элемент исчезнет со страницы (display: none)

Изменение других атрибутов элементов

attr(attrName) - получаем значение атрибута (например для использования в функции)
attr(attrName, attrValue) - устанавливаем значение атрибута.
removeattr(attrName) - удаляем атрибут.
Пример: получим альтернативное значение картинки (атрибут alt у img):
var altText = $('img').attr('alt');// в переменную запишется значение атрибута alt (="вот это")
$('img').attr('src','../images/logo.png');//изменяем атрибут src картинки

События

Событие - это любое действие пользователя (ввод данных, пролистывание страницы, клик мышки).
Ссылка на справочник всех событий jQuery: События
Пример: клик по ссылке вызовет alert(), но не переход на страницу ссылки:
$("a").click(function(event){
alert("Hello!");
event.preventDefault();//предотвратит переход на другую страницу
})
Если такое же событие применить к блоку, в котором расположена эта ссылка, то всплывет два alert() при нажатии на нее.
Это происходит потому что в DOM события начинают обрабатываться от родительского элемента к дочернему.
Чтобы событие вызывалось только дочерним элементом (ссылкой, но не блоком с таким же событием) используется метод event.stopPropagation();
$("a").click(function(event){
alert("Hello!");
event.preventDefault();//предотвратит переход на другую страницу
event.stopPropagation();//теперь родительский DOM элемент не вызовет свой event вместе с дочерним
})
Для сокращения можно вызвать сразу два эти метода (.preventDefault() и stopPropagation()) с помошью:
$("a").click(function(event){
alert("Hello!");
return false;//заменит два метода (.preventDefault() и stopPropagation()) сразу
})

В этом примере введенный пользователем в инпут символ дублируется в спан. В HTML имеем:
<input type="text" name="quantity" value="quantity">
<span id="msg"></span>
В скрипте добавим событие .keyup() нашему инпуту, которое реагирует на нажатие любой клавиши клавиатуры:
$('#quantity').keyup(function(){
let Value = $(this).val();
$('#msg').empty();
$('#msg').text(Value);
})
Введенный символ сохраним в переменную Value с помощью метода .val() и запишем его в спан методом .text(Value).

$(".anim").hide(500);

slow = 600мс, fast = 200мс

.animate()

Все анимации в jQuery крутятся вокруг метода .animate().
Данный метод берет одно или несколько свойств CSS и меняет его за одно или несколько итераций.
Количество итераций зависит от времени, но не реже одной итерации в 13мс.
Реализуем функции fadeIn() и fadeOut() с помощью animate():
$(".animTut").animate({
'opacity' : '0.5',//если указать после : 'hide' получится fadeOut, если 'show' то fadeIn
'height' : '50px',
'width' : '100px'//через запятую прописываем еще свойства для изменения размера блока
})

Нажми на меня

Очередность анимации можно менять несколько раз прописывая элементу метод .animate(), причем вторым параметром задавая очередь.
$(...).animate({'height':'+=50px'})
.animate({'width':'-=25px'},{queue:false})//queue обеспечит параллельный запуск анимации
Метод .stop() позволяет остановить выполнение последующих анимаций в очереди:
$(...).animate({'height':'+=50px'})
.stop();// проиграется только первая height анимация
.animate({'width':'-=25px'},{queue:false})

Для предотвращения всей анимации на странице (для отладки например) используется такая конструкция:
jQuery.fx.off = true;

After Before Append

Эти методы применяются для удаления или добавления элементов в DOM.
Метод appendTo() добавит элемент в указанное место на странице:
var $myDiv = $('<div id="my" class="page"> Hello! </div>').appendTo("body");//добавит div атрибутами прямо в конец body
Использование этого метода увеличит скорость загрузки, тк он обрабатывается методом parseHTML.
Чтобы ускорить загрузку, атрибуты элемента можно передавать вторым параметром:
var $myDiv = $('<div> Hello! </div>', {'id':'my', 'class':'page'}).appendTo("#content");
Также ускорить загрузку может использование метода attr() для указания в нем атрибутов:
var $myDiv = $('<div> Hello! </div>').attr({'id':'my', 'class':'page'}).appendTo("#content");

На чистом JS такое же действие выполняется не менее легко:
var myDiv = document.createElement('div');
myDiv.id = "my";
myDiv.className = "page";

.append() вставляет контент в конце выбранного элемента (в конце блока).
$("p").append("<span>LoremIpsum</span>");
Метод .prepend() и .prependTo работают аналогично append, но вставляют контент в НАЧАЛО блока.

AFTER и BEFORE

.after() вставляет контент после выбранного элемента(ов).
$("p").after("<hr/>");//поставит тег горизонтальной линии после ВСЕХ элементов "p"
Метод .insertAfter() позволяет поменять местами элементы в параметрах:
$("<hr/>").insertAfter("p");

.before() вставляет контент перед выбранным элементом(ми).
Аналогично работает и .insertBefore() - контент, метод, элемент перед которым вставлять:
$("$imgLogo").insertBefore(".header_menu");

.replace() .wrap() .cline() .detach()

Метод .replaceWith() заменяет выбранные элементы новыми:
$(".box1").replaceWith("<h2>New element</h2>");// заменит div.box1 на элемент h2
.replaceAll() делает тоже самое для всех выбранных элементов, но параметры меняются местами:
$("<h2>New element</h2>").replaceAll(".box");

Метод .wrap() оборачивает выбранный элемент внутрь другого элемента (<НовЭл> <СтарЭл> </НовЭл>):
$(".box1").wrap("<div class='new'> Текст </div>");//элемент .box1 окажется внитри элемента .new
.wrapAll() если применить в этом же примере (если у нас несколько .box на одном уровне), то он обернет их все в .new
.wrapInner() поступает по другому, он создает блок внутри выбранного блока, не трогая его контент:
$(".box1").wrapInner("<div class='new'> Текст </div>");
Получится так:
<div class="box1>"
<div class="new"> Текст </div>
Контент который был в box1
</div>
Также есть метод .unwrap() который удаляет родительский элемент выбранного элемента.

Метод .clone() копирует целый элемент для дальнейшей его вставки в DOM.
Он также позволяет скопировать и обработчики событий элемента.
$(".first").clone().appendTo(".third");//скопирует элемент .first и вставит в конец элемента .third

Метод .detach() удаляет элемент из DOM, но при этом сохраняет все данные о нем в jQuery.
Его следует использовать если надо удалить элемент, а потом вернуть его обратно при определенных условиях.
var par;//в переменную сохраним блок .detach__content
$("#btn__detach").click(function(){
if(par){
par.appendTo(".detach__test__box");
par = null;
} else{
par = $(".detach__content").detach();
}
});

.Offset, Position, Height, innerHeight

Метод .offset() позволяет вернуть позицию элемента относительно документа (значения left и top в пикселях):
var p = $(".first");//выбранный элемент сохраним в переменную
var offset = p.offset();//данные о прложении .first сохраним в другую переменную
p.html("Слева: " + offset.left + ", Сверху: " + offet.top);//запишем в наш .first элемент данные о его же положении
Также можно задать свои координаты оложения для элемента:
$(".box").offset({top:100,left:50});
Элементу .box добавится аблолютное позиционирование и расположение будет как указано в top и left.

Метод .position() действует похоже, но вернет позицию выбранного элемента относительно его родительского элемента (не страницы).
Метод .height() возвращает высоту элемента за вычетом отступов и границ.
Если в выборке несколько элементов, то он вернет высоту только первого, также возвращает он значение без единиц измерения.
В примере мы отобразим высоты объектов (p, document и window) в блоке ниже:
function showHeight(element, height){
$(".height__test__p").text("Height of the " +element+" is "+height);
};
$("#getP").click(function(){
showHeight("paragraph", $(".height__test__p").height());
});
$("#getD").click(function(){
showHeight("document", $(document).height());
});
$("#getW").click(function(){
showHeight("window", $(window).height());
});

 

Я просто параграф внутри блока .height__test__box, нажми на кнопку чтобы узнать мою высоту.

Метод .innerHeight() вернет высоту элемента включая его padding.
