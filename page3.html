Variables

There is no types in JS, all variables starts with "var"
var name;
You can add value with adding variable:
var myNumber = 12;
var myBool = true;
var myString = "Hello world";
Values can be added after creating the variable:
var numUser;
numUser = 1.05;

To show variable value on html page:
document.write("Now you see value on html" + myString)
