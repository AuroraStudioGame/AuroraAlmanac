Регулярные выражения

С помощью регулярных выражений можно производить проверку пользовательского ввода. К примеру на наличие определенных символов (@, _ ...).
В основном применяются при проверки полей форм (email).
Сначала создается переменная - паттерн с комбинацией символов для проверки:
var adr_pattern = /[0-9a-z_]+@[0-9a-z_]+\.[a-z]{2,5}/i;
В этом примере проверки email:
/ - означает что вводиться может множество символов, не только один
[0-9a-z_] - перечисление возможных символов (числа от 0 до 9 и латинские буквы от а до z, также символ _), которые могут находиться до знака "@"
+@ - после символов в [] должна стоять "собачка" - @
+\. - затем должна стоять одна единственная точка (\ - обозначает единство символа)
{2,5} - количество символов в [] перед! {} должно быть от 2 до 5
/i - означает что регистр текста не важен.

Для проверки регулярных выражений в формах используют функцию "test":
if(adr_pattern.test(email)==false){...}
Как альтернативный способ проверки email можно воспользоваться функцией "split":
if(email.split('@').length-1 == 0){...} - функция split разделяет строку (переменная email) на символы до @ и после @.
Функция length вернет в этом случае 1 и отнимаем от нее 1 и сравниваем с 0 - так можно проверить строку на наличие определенного символа (например @).

Таймеры

Похожи на циклы, которые выполняются с определенным периодом.
var inter = setInterval("myFunction1()", 1000); - функция setInterval присваивает заданный интервал выполнения чего-либо
Первый аргумент - код или функция, которая будет вызываться. Второй аргумент - промежуток времени, в который будет вызываться первый аргумент (в миллисекундах!)
var counter = 0;
function myFunction1(){
counter++;
alert("Секунд прошло " + counter);
if(counter==5) clearInterval(inter);}
Функция clearInterval() остановит выполнение функции myFunction1() после 5и ее выполнений.

Вторая важная функция "setTimeout()" - она вызывается один раз через заданный отрезок времени.
В отличие от setInterval, данная функция выполняется один раз с заданной задержкой (второй параметр)
setTimeout("timer1()", 1000); - каждую секунду функция ссылается на функцию timer1 и выполняет ее
С загрузки страницы прошло -1 секунд.

Принципы ООП

Стандартные классы: Math:
Класс "Math" содержит математические константы и функции. Обращаться можно так:
var pi = Math.PI; - присвоит переменной значение числа Пи
Примеры нескольких функций класса Math:
Math.abs(число) - вернет модуль числа
Math.sin(0.3) - вернет синус угла (параметр задается в РАДИАНАХ)
Math.random() - вернет случайное число от 0 до 1
Math.floor(5.3) - округляет значение до наименьшего (вернет 5)
Math.ceil(5.6) - округляет значение до наибольшего (вернет 6)

Стандартные классы: Date:
Содержит функции работы со временем.
var date = new Date(); - создаст объект класса Date, переменная будет содержать полное текущее время и дату
Далее чтобы узнать конкретную часть даты, используются функции:
var text = "Текущий год" + date.getFullYear(); - вернет только текущий год
...date.getMonth() - вернет текущий месяц
...date.getDate() - вернет текущий день недели
...date.getHours() - вернет текущий час дня
...date.getMinutes() - вернет текущее количество минут
...date.getSeconds() - вернет количество секунд на часах
Чтобы обратиться к какой-то конкретной дате, в параметрах конструктора Date необходимо указать эту дату полностью:
var date1 = new Date(2007, 0, 15, 22, 40, 10) - задает конкретную дату (год, месяц с 0!, день, час, минута, секунда)
Установить конкретное время можно и для каждого элемента даты(год, день, час...):
Для этого в функциях вместо "get" пишется "set"
date1.setMinutes(45);

Стандартные классы: Array:
var arr = new Array(3, 5, 9, 14, 27); - создаем объект класса, в качестве параметров - элементы массива
Чтобы узнать количество элементов в массиве используется функция "length":
text = "Длина массива" + arr.length;
С помощью метода "join" можно вывести все элементы массива:
arr.join(","); - вернет все элементы массива как строку, записанные через символ, заданный параметром
Сортировка массива выполняется методом "sort":
(arr.sort()).join(","); - вернет сортированную строку элементов массива по "возрастанию" по первому символу(тк join возвращает строку то по алфавиту)
Также существует метод "split" он создает массив, разделяя строковую переменную по заданному параметру.

Стандартные классы: String:
Для string не нужно создавать объект класса, он создается как обычная переменная:
var string = "Небольшая строка";
Чтобы узнать длину строки (количество символов) используется метод length:
string.length; - вернет количество элементов строчной переменной string
string.toUpperCase(); - вернет строку с символами в верхнем регистре
string.toLowerCase(); - вернет строку с символами в нижнем регистре
string.substring(1, 5); - метод для обрезки строки (в переметрах с какого элемента обрезать и до какого по номеру) этот метод переделывает строку в массив
string.charAt(3); - вернет 3ий символ строки (3ий по номеру как в массиве)
string.indexOf("шая") - вернет индекс (как в массиве) с которого в строке начинается сочетане символов (подстрока) "шая"
Если в метод indexOf параметром передать символы, не существующие в строке, то он вернет "-1". Это удобно для поведения каких-нибудь проверок.
