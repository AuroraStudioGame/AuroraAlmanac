box-sizing

-webkit-box-sizing -moz-box-sizing

Когда вы устанавливаете box-sizing: border-box; для элемента, поля и границы самого элемента(margin, border, padding) больше не увеличат его ширину.

.box1 {
width: 500px;
margin: 20px auto;
-webkit-box-sizing: border-box;
-moz-box-sizing: border-box;
box-sizing: border-box;}
У этого бокса нет отступов и границ, но он такой же длины

.box2 {
width: 500px;
margin: 20px auto;
padding: 50px;
border: solid blue 10px;
-webkit-box-sizing: border-box;
-moz-box-sizing: border-box;
box-sizing: border-box;}
У этого бокса есть отступы и границы, но длина такая же.

Поскольку так гораздо лучше, некоторые авторы хотят, чтобы все элементы на всех страницах всегда работали таким образом. Для этого нужно установить следующий CSS для страниц:
* {
-webkit-box-sizing: border-box;
-moz-box-sizing: border-box;
box-sizing: border-box;}
 Это гарантирует, что размеры всех элементов могут быть определены более интуитивно.
Поскольку box-sizing достаточно новое свойство, вы должны использовать -webkit- и -moz- префиксы, как я в этих примерах. Эта техника позволяет реализовывать экспериментальные возможности в конкретных браузерах. Кроме того, имейте в виду, что один из них это IE8 и старше.

Position:

Position: static;

static является значением по умолчанию. Элемент с position: static; отображается как обычно и не будет позиционироваться каким-либо специальным способом, он просто плюхнется вниз.

Position: relative;

relative ведет себя в точности как static но лишь до тех пор, пока вы не добавите некоторые дополнительные свойства.
Добавление свойств top, right, bottom, и left изменяет позицию элемента, сдвигая его в ту или иную сторону относительно его начального расположения. Другой контент не будет перестраиваться, чтобы заполнить любое свободное пространство, оставленное этим элементом.

.relative1 {
position: relative;}
.relative2 {
position: relative;
top: -20px;
left: 20px;
background-color: white;
width: 500px;}

Position: fixed;

Фиксированное позиционирование элемента относительно окна браузера таково, что он всегда занимает одно и тоже место даже при прокрутке страницы, словно он приклеен к экрану. Также как и для relative, дополнительные свойства top, right, bottom, и left для данного позиционирования будут работать.

.fixed {
position: fixed;
bottom: 0;
right: 0;
width: 200px;
background-color: white;}

Фиксированный элемент не оставляет за собой свободное пространство на странице, где он был расположен.

Position: absolute;

absolute наиболее сложное значение позиционирования. absolute ведет себя как fixed за исключением того, что позиционирование отсчитывается относительно ближайшего расположенного родителя а не относительно окна браузера. Если элемент абсолютно позиционирован и не имеет родителя,то он использует тело документа и продолжает двигатся вместе со всей страницей при скроллинге. Помните , что "позиционированный" элемент - это элемент с любым значением свойства position кроме значения static.

Этот элемент относительно позиционированный. Если бы элемент имел значение position: static; то его асболютно позиционированный ребенок убежал бы и стал позиционироваться уже относительно тела документа.

position: relative;
width: 600px;
height: 400px;

Этот элемент абсолютно позиционированный. Он позиционирован относительно его родителя.

position: absolute;
top: 120px;
right: 0;
width: 300px;
height: 200px;

Reset CSS

Эти стили помогут обнулить стандартные стили CSS для последующего переназначения.

Сначала обнулем поля и отступы:
body, html, div, blockquote, img, label, p, h1, h2, h3, h4, h5, h6, pre, ul,
ol, li, dl, dt, dd, form, a, fieldset, input, th, td{
margin: 0; padding: 0; border: 0; outline: none;}
 Сделаем заголовки одним размером для отображения в разных браузерах:
h1, h2, h3, h4, h5, h6{
font-size: 100%;}
 Далее определим поля и отступы для заголовков. Также удалим list-style-type из элементов списка. И установим базовый размер шрифта для элемента body.
body{
line-height: 1;
font-size: 88%;}
h1, h2, h3, h4, h5, h6{
font-size: 100%;
padding: .6em 0;
margin: 0 15px;}
ul, ol{
list-style: none;}
img{border: 0;}
 Добавим удобные классы для поплавков:
.floatLeft{
float: left;
padding: .5em .5em .5em 0;}
.floatRight{
float: right;
padding: .5em 0 .5em .5em;}

Также можно использовать готовые сбрки для обнуления стилей, загуглить "Reset CSS"
